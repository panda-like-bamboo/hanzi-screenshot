Screenshot Tool v0.1
====================

A full-featured desktop screenshot application.

Features:
- Global hotkey support (default: Ctrl+Shift+A)
- Area selection with magnifier
- Drawing tools: Rectangle, Ellipse, Arrow, Line, Dashed Line, Pen, Text, Mosaic
- Color picker and line width adjustment
- Save to file or copy to clipboard
- System tray integration
- Auto-start on system boot (optional)

Usage:
1. Press Ctrl+Shift+A to start screenshot
2. Drag to select area
3. Use drawing tools to annotate
4. Click Save or Copy button

Settings:
Right-click the tray icon and select Settings to customize hotkey, save path, and other options.
